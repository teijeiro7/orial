## Design Context

### Users
Usuario único (personal OS). Se abre en dos momentos: mañana para planificar el día, noche para revisar progreso. Contexto dark — pantalla en entorno de baja luz, uso rápido.

### Brand Personality
**Preciso · Crítico · Austero**

App de rendimiento personal. No es wellness ni lifestyle — es operativo, exigente, sin adornos. El usuario no quiere que la app le emocione; quiere que le informe con precisión.

### Aesthetic Direction
- Dark theme siempre. El usuario opera en entornos de baja luz.
- Flat cards (`GlassCard variant="flat"`). Glassmorphism = legacy, solo para overlays especiales.
- Tipografía: **Barlow Condensed** (BarlowCondensed-Bold/SemiBold) para números, fechas, encabezados de sección. **Manrope** (Regular/Medium/SemiBold) para cuerpo de texto, labels, captions.
- Paleta: `deepNavy` (#080C18) base, `surface` (#111827) para cards, `violet` (#7C3AED) como acento principal, `cyan` (#06B6D4) secundario. Usados con mesura.
- Anti-referencias: cualquier cosa "wellness" con gradientes suaves y tipografía redondeada. Cualquier look "SaaS dashboard 2023" con cards idénticas en cuadrícula.

### Design Principles
1. **Datos sobre decoración** — cada elemento debe justificar su espacio con información real.
2. **Jerarquía tipográfica fuerte** — Barlow Condensed para números grandes crea contraste inmediato con el cuerpo Manrope.
3. **Sin side-stripe borders** — prohibido `borderLeftWidth > 1px` como accento visual. Usar fondos tintados, dots, o diferenciación de opacidad.
4. **Sin glassmorphism por defecto** — GlassCard siempre en variant="flat". Glass solo para modales/overlays.
5. **Espaciado con ritmo** — secciones con paddingTop: 20, separadores con `borderBottomColor: border` (1px), no cards dentro de cards.
